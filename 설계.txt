# 블로그 만들기

- 자주 보는 형태
    - 내가 자주 보는 형태

### 나만의 일기장

1. html,css로 만든 메인 페이지를 node.js로 서버를 열어서 사용자에게 보여준다.
2. 메인 화면에서 post방식으로 제목, 날짜, 내용을 사용자가 입력한 것을 저장하라고 서버에 요청한다. 
3. 서버 node.js에서 POST방식으로 받은 제목, 날짜, 내용을 받아서 JSON형식으로 저장을 한다. 
4. JSON을 서브 node.js로 받아서 서브 페이지 html 파일을 만든다.
5. 서브 페이지html 파일에 미리 만들어둔 서브 페이지css를 적용해서  서버 node.js에 출력을 요청한다.
6. 서버에서 사용자에게 서브 페이지를 보여준다.

## 고민 - 지금 할 수 있을 것만 같은데

- 4번에서 기존 메인 페이지인 html에 있는 다음 페이지 버튼에 새로 만들어진 서브 html 파일로 연결을 해야 한다.
- 5번 html 을 만들 때 이전 페이지와 다음 페이지로 이동하는 것을 구현해야 한다. 다음 페이지는 구현이  될 것 같다는 생각이 드는데 이전 페이지 구성이 고민이 된다.

## +추가

1. 디자인 요소가 추가되어야겠다. -나중에 css만 변경하면 되는 부분
    1.  춘식이 느낌이나 색감을 사용해서 만들기(좋아하는 캐릭터)
    2. 제주도,부산 여행 갔을 때 사진을 활용해도 좋아 보인다.
        - 여행 갔을 때 일기 쓰는 게 재밌었던 기억이 너무 좋았고 그때 일기를 잘 못하고 한번에 지워져서 슬펐던 것이 기억나서
    3. 옛날 컴퓨터나 플로피디스크 모양이나 레트로 게임기 모양 등 옛날 전자기기류
        - 플로피디스크 포스트잇 모양처럼

2. 날짜를 하나하나 넣기 귀찮을 것 같다. 
    1. 버튼을 누르면 날짜 시간이 저장되는 형태
    2. 자동으로 저장할 때 날짜 시간이 들어가면 좋을 것 같다.
3. 다른 정보도 넣고 싶다.
    1. 사진 넣고 싶다 - 이미지 넣기
    2. 내가 갔던 길을 관련해서 넣을 수 없을까? (산책하면서 갔던 길을 기억하면 좋으니까)- 지도에 찍어서 어디서 어디까지 갔었는지 저장할 수 있게
4. 페이지 추가 - 달력처럼 놓고 무슨 일이 있었는데 10글자 내외로 글을 쓸 수 있고 누르면 안에 해당 날짜의 일기를 볼 수 있게 만들기
    1. 달력 페이지 에서 칭찬 스티커와 같이 무슨 일을 했을 때 부여할 수 있는데 이모티콘이나 스티커 같은데 것들이 있으면 좋아 보인다.
    2. 성과를 표시 하는 바도 좋아 보인다. 
        1. a , b청소 정도나 더러움 정도를 표시하면 좋겠다. 각각 해야 할 장소나 구역이 나와있고 그것을 바를 만들어서 더러움 정도를 내가 표기 할 수 있으면 좋겠다. - 보고 그 주말에 더러운 곳을  청소할 수 있으니까.
5. 체크 만들기 - 매일하는 것을 적어두고 체크 하는 게 좋아 보인다. 잊어버리지 않게
